# Praxis Mind - 個人化學習實踐中心

> 將抽象的學習計畫轉化為每日可執行的任務、可追蹤的進度、以及持續優化的回饋循環。

## 📱 應用概述

Praxis Mind 是一款專為知識工作者和終身學習者設計的 iOS 應用，旨在幫助用戶實踐個人化的學習方法，追蹤學習進度，並持續優化學習策略。應用融合了間隔重複、費曼技巧、行動學習等多種先進學習方法，並提供直觀的界面使這些方法容易實踐。

## 🌟 主要功能

- **目標儀表板**：視覺化呈現核心學習領域及進度
- **學習策略工具箱**：間隔重複卡片、費曼技巧練習區、類比與視覺化筆記
- **行動專案管理器**：任務驅動學習專案、MCP框架整合
- **效率加速器**：速讀計時器、關鍵詞摘要、情境記憶庫
- **元認知與潛意識日誌**：每日反思、情緒與專注力追踪、微任務生成器
- **整合與排程中心**：個人化學習時鐘、今日任務整合
- **回顧與分析**：每週學習回顧報告、知識連結視覺化

## 🛠️ 技術堆疊

- **前端框架**：SwiftUI & UIKit
- **架構模式**：MVVM (Model-View-ViewModel)
- **數據持久化**：Core Data & CloudKit
- **版本控制**：Git & GitHub
- **CI/CD**：GitHub Actions + Fastlane
